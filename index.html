LeVélo - Marseille

Vous cherchez un vélo ? Voila comment cela fonctionne ?

Etape 1

3 différentes icones indiquent la disponibilités et le nombre de vélos restants./p>

Etape 2

Pour connaitre l'état et le nombre de vélos, cliquez sur une icône sur la carte

Etape 3

Dans le panneau "Informations sur la station" à droite de la carte, cliquer sur "Réserver" afin d'ouvrir le panneau de réservation

Etape 4

Signer dans le panneau prévu à cet effet, cliquer sur "Effacer" pour recommencer si besoin, puis cliquer sur "Valider"

Etape 5

Votre réservation est valable pendant 20min avant confirmation, cliquer sur "Annuler" pour annuler votre réservation. Si vous décidez de réserver dans une autre station, cela annule automatiquement votre réservation précédente
